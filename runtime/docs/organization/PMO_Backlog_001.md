  envelope are normative would force an I2 redo (higher total cost).

  machinery is now sufficient and reusable; further org expansion is deferred; the
  next builds are engineering deliverables (Tasks 1–3), not more offices.

  blocks independent implementation and the Lock Review); formal semantics run in
  parallel as Verification-Office background work (Task 7), not on the critical path.
