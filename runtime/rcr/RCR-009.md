# RCR-009 — Live L1 conformance: Information Platform node (Connector, axis 1) (Runtime v1.1)

**Status:** Applied (v1.1) · **Type:** additive (extends `arves-conformance::live`; no existing
behavior/ABI change) · **Gate:** `cargo test --workspace` → **85 passed / 0 failed** (was 83; +2
tests). Freeze baseline re-advanced via `freeze_check.py update`.

## Motivation (strategic-program rank 7)

RCR-008 emitted the first live `ConformanceArtifact` for the **Kernel** node. The L1 pipeline
(`Information → Kernel → Query`) still lacked its first node — the **Information Platform**: a
`grep` for `impl Connector` returned zero. A stranger had a Kernel-behaviour target but no
canonicalization/provenance target.

## Change (what shipped)

- **`Connector`** (in `arves-conformance::live`): canonicalizes a `Source` into a deterministic,
  content-addressed `ProposedWrite` via the frozen ACS-002/001 codec — a canonical `uci.fact` body
  carrying the **five ontology aspects** (Identity=claim, Provenance=origin/source, Trust=confidence,
  Temporal=observed_at, TenantScope=tenant/workspace) + its ACS-001 ContentId. Same `Source` →
  same body → same address (idempotent).
- **`InformationPlatformProbe`** (impls the frozen `NodeProbe`): derives every check from the
  produced write — ORCH-004 (idempotent + a well-formed 34-byte `0x12 0x20` SHA-256 multihash),
  SHARD-001 (tenant-scoped), and the `ProvenanceTrustPresent` property.
- **`information_kernel_scenario()` + `run_information_kernel_scenario()`**: the L1
  **Information→Kernel** scenario observed end-to-end (Connector node + Kernel node), judged by the
  `LiveVerdictEngine` → the first **two-node** live `ConformanceArtifact` (`Verdict::Pass`).
- The `conformance_live` bin now runs it and prints a stable `LIVE-L1: PASS|FAIL` marker; the
  evidence probe row is updated (2 nodes, 6 invariant-checks Held). Tests:
  `live_l1_information_kernel_scenario_passes` + `connector_is_deterministic_and_addressed`.

## Scope (honest)

- **IS:** the Information Platform node is now live — real canonicalization with provenance/trust
  and an idempotent content address, observed into a two-node artifact. Behaviour L1 for the
  Information + Kernel nodes. Additive; `arves-conformance` already depended on `arves-acs`.
- **IS NOT (ranks 8/13):** the **Query** node (read-only WAL-replay projection) and the full
  end-to-end Information→Kernel→**Query** scenario that reads a committed truth back. Evidence-Level
  raise at grade **G0/G1**, NOT a G2 event.
