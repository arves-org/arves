ARVES-24: API Catalog v1.0

STATUS: API CONSTITUTION (AUTHORITATIVE CONTRACT INVENTORY)

Purpose

Define all external and internal API contracts across the ARVES platform.

API Philosophy

Tenant-Aware, Versioned, Observable, Secure and Composable APIs.

Platform APIs

Identity API, Tenant API, Workspace API, Policy API and Audit API.

Information APIs

Knowledge API, Ontology API, Knowledge Graph API, Metadata API and Connector API.

Cognitive APIs

Context API, Memory API, Reasoning API and Decision API.

Strategic APIs

Goal API, Planning API, Strategy API and Simulation API.

Experience APIs

Conversation API, Search API, Dashboard API, Notification API and Voice API.

Evolution APIs

Learning API, Benchmark API and Preference API.

Agent APIs

Agent API, Agent Runtime API, Agent Registry API and Agent Marketplace API.

Runtime APIs

Workflow API, Task API, Scheduler API and Execution API.

Canonical REST Pattern

GET, POST, PUT and DELETE patterns under /api/v1 resources.

Graph APIs

Entity Query, Relationship Query, Path Query and Graph Traversal.

Agent APIs Examples

Create Agent, Assign Goal, Pause Agent and Resume Agent operations.

Conversation APIs

Chat, Streaming Chat and Voice Conversation endpoints.

API Contract Template

Endpoint, Purpose, Request Schema, Response Schema, Authentication, Authorization, Events Produced, Events Consumed and Rate Limits.

Authentication

JWT, OAuth2, OIDC and API Keys.

Authorization

RBAC, ABAC and Policy Evaluation.

API Observability

Latency, Errors, Traffic and Consumer Metrics.

API Success Criteria

Stable, Versioned, Secure and Observable APIs.

Final Definition

API Catalog = Authoritative Contract Layer of ARVES.
