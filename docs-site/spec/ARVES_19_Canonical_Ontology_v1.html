ARVES-19: Canonical Ontology v1.0

STATUS: CANONICAL ONTOLOGY CONSTITUTION (AUTHORITATIVE KNOWLEDGE MODEL)

Purpose

Define the canonical entities, relationships, ownership rules, lifecycles and semantic meaning used across the entire ARVES platform.

Universal Relationship Types

OWNS, MEMBER_OF, WORKS_ON, ASSIGNED_TO, CREATED_BY, REPORTS_TO, DEPENDS_ON, SUPPORTS, RELATED_TO, LOCATED_IN, PART_OF, USES.

Person

Canonical Entity: Person

Definition: Authoritative semantic representation used across all cores.

Attributes: Identifier, Name, Description, Status, CreatedAt, UpdatedAt, Metadata.

Ownership: Tenant-scoped with explicit governance ownership.

Lifecycle: Created → Active → Updated → Archived.

Entity Resolution

Multiple representations of the same real-world object are resolved into a canonical entity.

All ontology definitions are versioned, governed and managed through the Ontology Registry.
