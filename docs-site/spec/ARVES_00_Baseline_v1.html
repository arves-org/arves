Part 1 - Purpose

Make the scope of ARVES v1.0 indisputable: what is complete, what is consciously deferred, and what is a candidate for the next major version. This is the reference point the project returns to.

Part 4 - Aspirational (north-star, not v1.0 commitments)

Present in the corpus as vision, explicitly NOT part of the v1.0 normative baseline:

  • L3/L4 capabilities: recursive self-improvement, autonomous strategic evolution (Vol 11).
  • Embodied/physical intelligence at scale: robotics, drones, vehicles (Vol 8).

The mindset shifts from "what is the correct architecture?" to "how do we run it at scale?"

Part 6 - Baseline Rule

ARVES v1.0 scope is closed. Additions are deferred to v2 or handled as amendments per the Reference Lifecycle. The Specification Era does not reopen.
